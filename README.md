## About Me

I'm a **.NET & Fullstack Software Engineer** based in **Auckland, New Zealand**, with solid experience building and maintaining scalable web systems across the full stack.

Currently focused on:
- **Back-end development** with C#, .NET 8, ASP.NET, and NestJS
- **Front-end** using Angular and Vue.js
- **Cloud & DevOps** with Azure DevOps, GitHub Actions, AWS, and Terraform
- **Distributed systems** with RabbitMQ, Redis, and MongoDB

Postgraduate in **DevOps & Continuous Software Engineering** (PUC Minas, 2025) and B.Sc. in **Information Systems** (PUC Minas, 2023).
